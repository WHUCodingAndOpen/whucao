2023新生购机指南 ​

TIP

本指南为 计院青协 & 微软学生俱乐部 共同撰写，如果你对本指南有建议，欢迎在 WHUCAO 上面 PR 我们！

TIP

本文档主要面向大部分对于购机比较陌生的小白，故而主要对于购买笔记本电脑的选择进行讨论（二手电脑和台式机购买暂不做讨论）

购买渠道 ​

  1. 不要选择线下电脑城！不要选择线下电脑城！即使有“熟人”也不要选择线下电脑城！电脑城套路太多、购买的每一步都会想方设法获得更多利润。所以一定要选择线上购买渠道
  2. 目前主流线上渠道有：京东、淘宝、拼多多等。在这些渠道购买时，不要盲目地只搜索目标型号笔记本，一定要看准商品的商家名称。一般来说，在京东：<品牌名>+京东自营官方旗舰店（认准“自营官方”）；在淘宝：<品牌名>+官方旗舰店（认准有”天猫“标识），这些店一般都是由品牌官方运营，商品质量、售后服务都有保证。除此之外的店铺一般都为第三方运营，不推荐在这种店购买，很容易出现买来的产品是二手、上一代的情况，售后也得不到保障。鉴于许多品牌在拼多多没有自己运营的旗舰店，虽然在价格上有一定的优惠，但是一般都是老一代机型、退换机等情况。总的来说，尽量在各品牌自己运营的京东的自营官方旗舰店、淘宝的天猫官方旗舰店购买笔记本！

需求分析 ​

针对计算机专业学生对于电脑的通用性需求进行分析

计算机专业学生的电脑常见使用场景：

  1. 代码编辑/项目开发（重型ide)

    对应需求：cpu，内存，硬盘（多套开发工具挺吃存储空间的）

  2. 浏览器同时打开海量标签页（检索问题/...）

    对应需求：内存

  3. 本地ai部署或训练

    对应需求：nvidia gpu

  4. 打游戏

    对应需求：gpu，硬盘，cpu，内存

  5. pr、达芬奇等专业视频图像编辑软件（爱好）

    对应需求：cpu，内存

  6. ...

TIP

总而言之, 因为专业的需要, 为了能获得在日后的专业学习以及自身兴趣爱好的探索中收获较好的体验, 建议大家重视自己第一台电脑的性能与可靠性.

不可能三角：价格，轻便，性能 ​

  1. 如果你更需要轻薄、更高续航、可以轻松随身携带的笔记本，且没有游玩大型游戏的需求、未来几年不接触ai相关方向的学习。你应该选择轻薄本。
  2. 如果你即想要有足够的游戏体验（即有使用独立显卡的需求）、又想拥有一定的便携性。你应该选择全能本。
  3. 如果你不经常携带笔记本外出（带笔记本去教室、图书馆），不太介意笔记本的厚重；有玩游戏、学习ai等这些使用独立显卡的需求。你应该选择游戏本。

TIP

现在没有一些需求不代表以后不会产生这些需求，比如你后来逐渐想追求更高的游戏体验等等，所以要把未来可能出现的需求考虑进去

笔记本基本参数 ​

当我们在挑选一款笔记本时，我们应该看如下的这些参数来评估它是否满足我们的需求

硬盘/内存 ​

TIP

硬盘这里指使用nvme协议的M.2接口的固态硬盘，注意内存与硬盘的区别：内存指电脑的运行内存

  1. 通常的笔记本电脑默认标配的512gb固态硬盘大部分情况是不够使用的，因此
    1. 若板载硬盘，则选购至少1t以上版本
    2. 若单m.2硬盘接口（戴尔笔记本较多），建议选购1t以上版本，否则后期升级硬盘时设计系统迁移，比较麻烦
    3. 若双m.2硬盘接口（大部分游戏本），则可以只买512gb版本，自行选购硬盘加装（有的黑心品牌加硬盘收费，说的就是戴尔）
  2. 硬盘接口规格 pcie3.0 4.0为现行主流配置，都满足日常使用（规格越高读写速度越快）
  3. 内存规格ddr3, 4, 5都满足日常使用（规格越高读写速度越快）
  4. 内存16gb满足绝大部分适用场景，因人而异有可能产生不够用的情况（比较少，比如我），因此建议选择非板载内存的机子（可以自行更换内存条），因为你也不知道自己会不会有这方面的需求，多一份选择

Cpu ​

  1. Intel or amd?
    1. 通常来说性能相近，对标层次相同的intel和amd cpu，intel会在如pr，ps，blender等生产力软件上的表现更好一些
    2. 就能耗比而言amd表现更好一些，自带核显性能更强，在显示能力方面已经接近独显，不带独显或者对于续航有追求的可以选择amd（轻薄本选择amd更有优势）
  2. amd锐龙r7, r5系列，intel酷睿p系列或者h系列i7，i5及以上都是满足计算机专业的基本需求的，更好的cpu或许可以为你带来更好的编译速度等，但不是刚需

Gpu ​

TIP

GPU即显卡, 最初的诞生是为了承担图像显示相关的计算功能(说人话就是打游戏), 与cpu的通用计算能力不同, gpu专精于并行计算, 因而在人工智能发展迅速的今天, gpu也被广泛当作共人工智能训练和部署使用的计算资源.

  1. gpu这里特指独立显卡，简称独显，主要有nvidia以及amd两家，amd独显比较少见，主要见于rog幻系列，计算机专业的同学可能需要使用到gpu进行ai本地部署或训练（当然大部分情况下ai部署和训练都是在服务器上完成的），这个需求下基本只能选择nvidia显卡（cuda）
  2. 一般带有独立显卡的机子大部分都会有一些厚重，本地ai部署与训练也非刚需，如果在意轻便的话可以放弃独显（或者加钱）
  3. 选择带有nvidia gpu的机子至少gtx1650往上，rtx30系以及40系都可以选择，是财力而定，追求高配是不必要的（除非追求流畅3a）

外设 ​

TIP

外设是指除开电脑主机以外的其他配件, 即屏幕, 键盘, 触控板等

  1. 屏幕素质，作为计算机专业学生，屏幕是你将要长时间观看的，建议大家重视屏幕的素质，一般来说1080p可以满足基本需要，有能力建议2k（现在2k基本上都是标配，很多品牌都没有1080p，除了天选），屏幕亮度尽量不要低于350nit，尤其注意色彩表现，有的廉价机子可能会在屏幕上减配（色彩表现很差的话会瞎眼）
  2. 键盘尽量选择带有全高方向键的（写代码刚需（vim用户除外）），一般来说小键盘不是刚需
  3. 接口需求（因人而异）：外接显示器(hdmi/dp/雷电3)，外接鼠标键盘(usbA)，插网线(光口)，如果轻薄本都没有则需要自行购买拓展坞，会增加一些麻烦.

TIP

  • 如果想要充分了解一款笔记本的性能参数，可以通过以下几种途径了解：
    • b站相关评测视频
    • 官网介绍

品牌/产品优势分析 ​

笔记本电脑的品类主要可以分为：游戏本（性能本），全能本（全都不能本，如能），轻薄本，二合一（平板+笔记本），windows掌机（?）

TIP

总的建议: 买新不买旧

游戏本 ​

cpu以及独显均高配，性能较好的同时能耗较高，牺牲了续航和轻便（部分高端游戏本除外）

代表：

  1. 高端游戏本：rog系列（败家之眼形态），雷蛇，外星人，微星系列等
  2. 大众游戏本（一线大牌）：rog系列（持家之眼形态），华硕天选，戴尔游匣，惠普暗影精灵/光影精灵，宏碁暗影骑士
  3. 呃呃：拯救者（r9000系列还行）
  4. 非一线但近几年进步较大的游戏本：机械革命蛟龙、红米G

选购建议：结合以上的需求分析和自己的预算，提前在b站查找相关笔记本的评测视频。一线品牌线下售后点比较多，方便快速售后维修，非一线品牌价格更有优势，但售后点少，需要邮递，时间长且容易出现损坏。

全能本 ​

比较轻薄的机身厚度、比较轻便的机身重量、比较强劲的机器性能、比较够用的机器续航

代表：

惠普战99、华硕无畏、rog幻

选购建议：因为其定位比较模糊等原因，市场上可供选择的全能本少。配置的质量和多样性不如同价位的游戏本、多加考虑。

轻薄本 ​

本如其名，大众商品的价格比较便宜, 同时也不乏高端(an gui)轻薄本的存在

代表：

MacBook、ThinkBook、matebook、magicbook...

选购建议：有更多的续航、游戏需求的选择amd笔记本；有更多视频图像创意需求的选择intel笔记本。笔记本作为电子产品，通常一年一换代，轻薄本换代的主要区别是cpu的换代。22到23，同系列的cpu提升基本可以忽略不计了，所以很多2022款的轻薄本在性能相似的前提下有更实惠的价格。（注意：仍然要在官方运营的旗舰店购买2022款，不要在其他第三方店购买老款笔记本）。如果你是华为/荣耀用户，建议购买华为/荣耀的轻薄本，可以享受华为/荣耀生态的便捷功能。

二合一 ​

代表： 微软Surface pro，华为matebook e

选购建议：通常高价低配，但是便携性拉满，两种形态可以一定程度上满足一些平板使用的需求（记笔记等），但如果作为自己唯一的电脑，建议谨慎选择并仔细甄别参数配置，如果想在二合一上拥有等同一般笔记本的性能，得加（很多）钱。同时也需要接受屏幕较小，接口极少，生产环境下续航拉跨的缺点。

windows掌机 ​

全新的笔记本类别, 确实是算便携式的windows系统电脑, 如果喜欢特立独行, 别具一格, 并且完全不在意电脑除游戏以外的使用场景的人可以选择.

省流 ​

  • 尽量在快递员在场的情况下当场检查包装是否完整，如果磕碰十分明显可以拒签

  • 开箱以及第一次启动尽量全过程拍摄，如果有问题可以直接无理由退换，留下视频证据方便和客服拉扯

  • 开箱前检查封口贴纸是否完整以及箱子上的配置信息是否正确

  • 开箱后检查配件是否齐全

  • 开机前检查电脑上是否有明显划痕或者使用痕迹

  • 开机时跳过联网激活（已激活的机子不支持无理由退换），使用图吧工具箱依次检查：

  1. 硬盘使用次数
  2. 硬件信息
  3. 烤机（检查cpu/gpu稳定性，避免矿机（虽然今年很少见））
  4. 检查屏幕亮度均匀性以及是否有坏点
